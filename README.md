# sap-rap-travel-management System
SAP RAP Travel Management System developed using ABAP Cloud, CDS View Entities, OData V4, Fiori Elements, and Behavior Definitions.

## Overview

The SAP RAP Travel Management System is a RESTful ABAP Programming Model (RAP) application developed using SAP ABAP Cloud.

This project demonstrates the implementation of a complete RAP Business Object using CDS View Entities, Behavior Definitions, OData V4 services, and Fiori Elements following SAP Clean Core principles.

The application allows users to manage travel records through a modern SAP Fiori Elements user interface.
